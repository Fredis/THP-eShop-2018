### Installing

$	Git init <br>
$	Git clone https://github.com/Fredis/THP-eShop-2018 <br>
$	Bundle install <br>
$	Rails db:create<br>
$	Rails db:migrate<br>
$	Rails db:seed<br>
$	Rails server <br>
$	Localhost (http://localhost:3000)
